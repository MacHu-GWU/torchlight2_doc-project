Install
------------------------------------------------------------------------------

``torchlight2_doc`` is released on PyPI, so all you need is:

.. code-block:: console

    $ pip install torchlight2_doc

To upgrade to latest version:

.. code-block:: console

    $ pip install --upgrade torchlight2_doc
