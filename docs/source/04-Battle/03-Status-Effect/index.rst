Status Effects 状态效果
==============================================================================

游戏中有多种负面状态

- 燃烧: 受到持续的火焰伤害
- 冰冻: 移动, 施法, 攻击速度降低 33%
- 中毒: 造成的所有伤害 和 护甲值 降低 33%
- 电击: 受到攻击时会放出闪电球对周围的敌人造成伤害
- 流血: 受到持续的流血伤害, 该伤害既不是物理伤害也不属于元素伤害, 但会受到智力的加成
- 致盲: 可视范围大幅降低, 无法发现位于附近的玩家.
- 拖曳: 将玩家拖向怪物所在位置
- 逃跑: 受到惊吓逃跑, 此时不会主动攻击和施放技能.
- 定身: 无法移动, 但可以攻击和施法.
- 击退: 被击退
- 昏迷: 无法进行任何行动
- 减速: 降低 移动 或 施法 或 攻击速度.
