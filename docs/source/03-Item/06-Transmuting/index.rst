Transmuting 合成
==============================================================================

和 D2 类似, 在 Act2 中会有一个合成大师 NPC, 能将多个物品转化成其他的物品. Act3 和 地图工房 中也有这个 NPC.

下面是所有的配方列表. 其中有用的有:

- A Random Unique 洗唯一物品
- Add Socket to Item 打孔公式
- High Grade Socketable 洗镶嵌物
