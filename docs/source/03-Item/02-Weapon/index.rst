武器的分类
------------------------------------------------------------------------------

单手武器:

- 单手剑
- 单手斧
- 单手锤
- 拳套

单手远程武器:

- 手枪
- 魔杖

双手武器:

- 双手剑
- 双手斧
- 双手锤
- 长柄武器

双手远程武器:

- 散弹枪
- 加农炮
- 弓
- 十字弓
